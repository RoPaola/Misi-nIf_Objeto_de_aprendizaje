La Misión IF: Código en la Nave Galáctica

Aprende condicionales mientras salvas una nave espacial

Introducción

Sos un cadete en una nave Galáctica. Para completar tu misión, necesitas usar estructuras condicionales y tomar decisiones críticas para el sistema de abordo.

¿Qué es una condicional?

La sentencia if en programación es una estructura de control condicional que permite ejecutar un bloque de código solo si se cumple una condición lógica específica . Si la condición es verdadera, se ejecuta el código dentro del bloque if; de lo contrario, se omite ese bloque y el programa continúa.

Se inicia con la palabra reservada if.

La condición que se evalúa como verdadera (true) o falsa (false) pueden incluir comparaciones :

      Mayor que >
      Mayor o igual >=
      Menor que <
      Menor o igual <=
      Igual que ==
      Distinto que !=

Por ejemplo, en pseudocódigo:

si (oxigeno < 20%) {
    activarProtocoloEmergencia();
} else {
    continuar();
}

Misión 1: Nivel de oxígeno

Si el nivel de oxígeno es menor a 20%, activá el protocolo de emergencia. Si es mayor a 50%, activá los ventiladores.

Misión 2: Simulación de Crisis Galáctica

Formando un equipo de 2 o 3 cadetes. Analicen y resuelvan al menos una de las siguientes situaciones críticas utilizando estructuras condicionales y múltiples variables. Presenten su solución en pseudocódigo o código real, y justifiquen su razonamiento.

Situación 1: Fuga de oxígeno

  • oxígeno (porcentaje)
  • atmósferaEstable (true o false)
  • puertaExteriorCerrada (true o false)

Si el oxígeno es menor a 25% y la atmósfera no está estable, activar protocolo de emergencia. Si la puerta exterior está abierta, cerrarla primero.

Situación 2: Gravedad alterada

  • gravedad (valor entre 0 y 10)
  • motoresActivos (true o false)
  • tripulantesDespiertos (true o false)

Si la gravedad es menor a 3 y los motores están activos, aumentar potencia. Si los tripulantes están dormidos, emitir alerta por intercomunicador.

Situación 3: Falla de energía

  • bateria (porcentaje)
  • modoEmergencia (true o false)
  • panelSolarActivo (true o false)

Si la batería está por debajo del 15%, activar el modo emergencia. Si el panel solar está activo, derivar energía prioritaria a soporte vital.
